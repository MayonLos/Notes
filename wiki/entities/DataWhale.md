> **一句话描述**：DataWhale 是一个专注 AI 领域的开源组织，以"for the learner，和学习者一起成长"为愿景，构建面向学习者的开源学习社区。

## 关键属性

| 属性 | 值 |
|:---|:---|
| 类型 | 开源组织 |
| 领域 | 人工智能教育 |
| 愿景 | for the learner，和学习者一起成长 |
| 协作模式 | 组队学习 + Forking 工作流 |

## 代表性项目

- **[[摘要-thorough-pytorch|深入浅出PyTorch]]** — PyTorch 从入门到实战的系统教程，配套 B 站视频
- **easy-rl** — 强化学习教程（仓库中提及）

## 协作方式

- 使用 GitHub Forking 工作流进行内容协作
- 课程以 Markdown + Jupyter Notebook 形式维护
- 组队学习模式：设定学习周期，成员共同推进
- Commit Message 规范：`<type>: <short summary>`（如 `[docs #NO]:others`）
