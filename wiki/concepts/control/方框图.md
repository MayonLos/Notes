> **一句话定义**：用图形符号表示控制系统各环节传递函数及其连接关系的图示工具，可通过等效变换化简为单一传递函数。

> [!tip] 学习路径
> **前置**：[[Laplace变换]]（传递函数的数学基础）、[[物理建模]]（建立各环节的 $G(s)$）
> **本页**：方框图——系统结构的图形化表示与化简
> **后续**：→ [[时域响应]]（闭环传函确定后分析时域性能）

---

## 基本图形元素

| 元素         | 符号             | 含义                     |
| :--------- | :------------- | :--------------------- |
| **方框（环节）** | $\boxed{G(s)}$ | 一个子系统，输出 = $G(s)$ × 输入 |
| **信号线**    | →              | 携带信号（Laplace 变换后的变量）   |
| **比较点**    | ⊕              | 信号相加/相减，可有多路输入         |
| **分支点**    | •              | 同一信号引出到多处，值不变          |

> [!important] 方框内只写传递函数 $G(s)$，不写时域表达式。

---

## 三种基本连接

### 串联（级联）

$$G_1(s) \rightarrow G_2(s) \rightarrow \cdots \rightarrow G_n(s)$$

$$G_{等效}(s) = G_1(s)\cdot G_2(s)\cdots G_n(s)$$

各环节传递函数**直接相乘**。

---

### 并联

多个环节共享同一输入，输出相加：

$$G_{等效}(s) = G_1(s) \pm G_2(s) \pm \cdots \pm G_n(s)$$

---

### 反馈（最重要）

```
R(s) →⊕→ E(s) → G(s) → C(s)
        ↑                  |
        └──── H(s) ←───────┘
              (负反馈)
```

| 变量     | 含义                |
| :----- | :---------------- |
| $R(s)$ | 参考输入（给定值）         |
| $E(s)$ | 误差信号 $= R - B$    |
| $G(s)$ | 前向通道传递函数          |
| $H(s)$ | 反馈通道传递函数          |
| $B(s)$ | 反馈信号 $= H(s)C(s)$ |
| $C(s)$ | 输出                |

**闭环传递函数**（负反馈）：

$$\boxed{\Phi(s) = \frac{C(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}}$$

> [!important] 推导
> $C = GE = G(R - HC)$，展开得 $C(1+GH) = GR$，故 $\Phi = G/(1+GH)$。
>
> 当 $H(s) = 1$（单位负反馈）时：$\Phi(s) = \dfrac{G(s)}{1+G(s)}$

**开环传递函数**：$G_{\text{open}}(s) = G(s)H(s)$（断开反馈后的前向×反馈）

## 等效变换规则

化简复杂框图时，逐步应用以下规则：

### 相加点的移动

| 操作 | 规则 |
|:-----|:-----|
| 相加点**越过方框前移**（向输入侧移）| 在移动支路上**串入** $\dfrac{1}{G(s)}$ |
| 相加点**越过方框后移**（向输出侧移）| 在移动支路上**串入** $G(s)$ |

**相加点前移**（扰动在 G 之后 → 移到 G 之前，需在扰动支路串 $1/G$）：

```tikz
\begin{tikzpicture}
\tikzset{
  blk/.style={draw, rectangle, minimum height=2em, minimum width=3em},
  sm/.style={draw, circle, minimum size=1.5em, inner sep=0pt}
}
\node (x1) at (0,0) {$X$};
\node[blk] (G1) at (1.8,0) {$G$};
\node[sm]  (s1) at (3.4,0) {$+$};
\node (y1) at (4.6,0) {$Y$};
\node (d1) at (3.4,1.4) {$D$};

\draw[->] (x1)--(G1);
\draw[->] (G1)--(s1);
\draw[->] (s1)--(y1);
\draw[->] (d1)--node[right]{$+$}(s1);

\node at (5.5,0) {$\Rightarrow$};

\node (x2) at (6.4,0) {$X$};
\node[sm]  (s2) at (7.8,0) {$+$};
\node[blk] (G2) at (9.6,0) {$G$};
\node (y2) at (10.8,0) {$Y$};

\draw[->] (x2)--(s2);

\draw[->] (G2)--(y2);
\draw[->] (d2)--(Gi);
\draw[->] (Gi)--node[right]{$+$}(s2);
\end{tikzpicture}
```

### 分支点的移动

| 操作 | 规则 |
|:-----|:-----|
| 分支点**越过方框前移** | 在移动支路上**串入** $G(s)$ |
| 分支点**越过方框后移** | 在移动支路上**串入** $\dfrac{1}{G(s)}$ |

**分支点前移**（分支点在 G 之后 → 移到 G 之前，需在支路串 $G$）：

> [!warning] 易错
> 相加点和分支点的移动方向与补偿方向**正好相反**，容易混淆。记忆口诀：
> - **相加点前移** → 支路加 $1/G$（越过去要"除以"那个框）
> - **分支点后移** → 支路加 $1/G$（越过去要"除以"那个框）
> - 两者后移/前移的补偿方式**互为倒数**。

### 相加点互换

同一信号的相加点可以**任意交换顺序**（加法交换律）。

## 化简步骤

> [!tip] 通用流程
> 1. 标出所有信号变量（$R, E, C$，以及中间节点）
> 2. 先处理**内环**（最内层反馈回路），化简为单框
> 3. 再处理**外环**，重复直至化为单框
> 4. 若有交叉支路，先移动相加点/分支点消除交叉

---

## 例题

### 例 1：单位负反馈系统

$$G(s) = \frac{10}{s(s+1)}, \quad H(s) = 1$$

$$\Phi(s) = \frac{G}{1+G} = \frac{\dfrac{10}{s(s+1)}}{1+\dfrac{10}{s(s+1)}} = \frac{10}{s^2+s+10}$$

---

### 例 2：含内环的双回路系统

```
R →⊕→ G₁ →⊕→ G₂ → C
    ↑        ↑        |
    │        └─ H₂ ←─┤
    └────── H₁ ←──────┘
```

**步骤 1**：化简内环

$$G_{内}(s) = \frac{G_2}{1 + G_2 H_2}$$

**步骤 2**：化简外环（$G_1$ 与 $G_{内}$ 串联后作为前向通道）

$$\Phi(s) = \frac{G_1 G_{内}}{1 + G_1 G_{内} H_1} = \frac{G_1 G_2}{(1+G_2 H_2) + G_1 G_2 H_1}$$

---

## 误差传递函数

对于标准负反馈系统：

$$\Phi_e(s) = \frac{E(s)}{R(s)} = \frac{1}{1 + G(s)H(s)}$$

注意：$\Phi(s) + \Phi_e(s) \cdot H(s) = 1$（当 $H=1$ 时，$\Phi + \Phi_e = 1$）
