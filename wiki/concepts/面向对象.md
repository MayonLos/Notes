> **一句话定义**：以对象为中心的编程范式，核心四大原则为抽象、封装、继承、多态，是构建大规模可维护程序的基础。

> [!tip] 学习路径
> **前置**：[[基础语法]]（指针、引用、函数）
> **本页**：OOP——类、继承、多态、虚函数、抽象类
> **后续**：→ [[标准模板库]]（泛型容器与 OOP 设计思想）→ [[内存管理]]（析构函数、RAII 的 OOP 实现）

## 四大原则

| 原则 | 含义 |
|:---|:---|
| **抽象** | 隐藏实现细节，暴露本质特征 |
| **封装** | 隐藏内部状态，提供公共接口 |
| **继承** | 代码复用，建立类层次结构 |
| **多态** | 统一接口，多样实现 |

## 类与对象

```cpp
class Animal {
public:
    std::string name;
    void eat() { std::cout << name << " 在吃\n"; }
protected:
    int age;        // 子类可访问，外部不可
private:
    int energy;     // 仅类内可访问
};
```

### 构造函数与析构函数

```cpp
class Person {
private:
    std::string name;
    int age;
public:
    Person() : name("未知"), age(0) {}
    Person(const std::string& n, int a) : name(n), age(a) {}
    ~Person() { std::cout << "Person " << name << " 销毁\n"; }
};
```

> [!warning] 初始化列表顺序
> 初始化列表的执行顺序遵循**成员声明顺序**，而非书写顺序。

## 访问修饰符

| 修饰符 | 类内 | 派生类 | 类外 |
|:---|:---|:---|:---|
| `public` | ✅ | ✅ | ✅ |
| `protected` | ✅ | ✅ | ❌ |
| `private` | ✅ | ❌ | ❌ |

## 继承与多态

```cpp
class Animal {
public:
    virtual void sound() { std::cout << "...\n"; }  // 虚函数
    virtual ~Animal() = default;                     // 虚析构
};

class Dog : public Animal {
public:
    void sound() override { std::cout << "Woof!\n"; }
};

// 运行时多态
Animal* a = new Dog();
a->sound();   // 输出 "Woof!"（动态绑定）
delete a;
```

> [!important] 虚析构函数
> 多态基类**必须**声明虚析构函数，否则通过基类指针 `delete` 时不会调用派生类析构函数，导致内存泄漏。

## 抽象类与接口

```cpp
class Shape {
public:
    virtual double area() = 0;   // 纯虚函数 → 抽象类
    virtual ~Shape() = default;
};

class Circle : public Shape {
    double r;
public:
    Circle(double r) : r(r) {}
    double area() override { return 3.14159 * r * r; }
};
```

> 抽象类不能实例化，必须由派生类实现所有纯虚函数。

## 运算符重载

```cpp
class Vector2D {
public:
    double x, y;
    Vector2D operator+(const Vector2D& other) const {
        return {x + other.x, y + other.y};
    }
    bool operator==(const Vector2D& other) const {
        return x == other.x && y == other.y;
    }
};
```
