> **一句话定义**：锁存器和触发器是具有**记忆性**的双稳态存储单元，是顺序逻辑电路的基本构件——输出不仅取决于当前输入，还取决于历史状态。

![[assets/illust/锁存器和触发器.png]]

## 与组合逻辑的本质区别

| | 组合逻辑 | 时序逻辑（触发器）|
|:---|:---|:---|
| 输出依赖 | 仅当前输入 | 当前输入 + **历史状态** |
| 反馈 | 无 | 有（输出回接输入）|
| 稳定状态 | 唯一 | 双稳态（0 或 1）|

---

## 一、基础 SR 锁存器（NAND 门实现）

两个 NAND 门**交叉耦合**，形成闭合正反馈回路：

$$\boxed{Q^{n+1} = S_d + \overline{R_d} \cdot Q^n} \quad \text{约束：} S_d \cdot R_d = 0$$

```
S̄d ──┤&├── Q
      ↕  ↕
R̄d ──┤&├── Q̄
```

### 状态转换真值表

| $\overline{S_d}$ | $\overline{R_d}$ | $Q^{n+1}$ | 说明 |
|:---:|:---:|:---:|:---|
| 0 | 1 | 1 | **置位**（Set）|
| 1 | 0 | 0 | **复位**（Reset）|
| 1 | 1 | $Q^n$ | **保持** |
| 0 | 0 | ✗ | **禁止**（两端均为1，违反互补）|

> [!warning] 禁止态
> $\overline{S_d}=\overline{R_d}=0$ 时，两个NAND门输出均被强制为1，违反 $Q$ 与 $\overline{Q}$ 互补关系。撤除激励后状态不确定。

### 三态SR锁存器（CMOS传输门实现）

在基础SR锁存器的输入端加入**传输门（TG）**，由使能信号 EN 控制通断：

- EN=1（TG导通，ccMOS）：锁存器透明，输入可改变状态
- EN=0（TG截止，CMOS互补门锁住）：锁存器保持，输入被隔离

这是门控锁存器的 CMOS 物理实现基础。

## 二、门控锁存器（电平触发）

### 门控 SR 锁存器

在基础SR输入端加入 CP 控制：

$$\text{CP=0}: Q^{n+1}=Q^n \quad \text{CP=1}: Q^{n+1}=S+\overline{R}Q^n,\; SR=0$$

> [!warning] 透明性问题
> CP=1 期间，锁存器完全透明——输入 S/R 的任何变化都即时反映到输出。这是电平触发器件的根本缺陷。

### 门控 D 锁存器

令 $S=D,\; R=\overline{D}$，消除禁态约束：

$$\text{CP=1}: Q^{n+1}=D \qquad \text{CP=0}: Q^{n+1}=Q^n$$

**D锁存器 vs SR锁存器 比较：**

| | 门控SR锁存器 | 门控D锁存器 |
|:---|:---|:---|
| 输入端 | S、R 独立 | 单端D |
| 禁态 | 有（SR=1 禁止）| 无 |
| CP=1期间 | 跟随S/R | 跟随D（透明）|
| 优势 | 独立控制置/复位 | 简单，无禁态 |
| 劣势 | 需保证SR约束 | 仍透明，有一次采用隐患 |

## 三、主从触发器与"一次采用现象"

### 主从 SR 触发器

两级门控SR锁存器串联，CP 反相控制：

- CP=1：主触发器透明（接收输入），从触发器锁住（输出不变）
- CP=0：主触发器锁住，从触发器翻转，输出更新 → $Q^{n+1} = Q'$

$$Q^{M+1} = Q' + \overline{Q'} \cdot Q^n$$

> [!warning] 一次采用现象（One-Catch Problem）
> CP=1 **整个高电平期间**，主触发器都处于透明状态。若输入在此期间多次变化（如噪声干扰），主触发器会持续响应，最终锁住的是**最后一次变化**的值，而非 CP 上升沿时刻的值。

### 主从 JK 触发器

$$Q^{n+1} = J\overline{Q^n} + \overline{K}Q^n \quad (\text{CP}\downarrow \text{ 时输出翻转})$$

| $J$ | $K$ | $Q^{n+1}$ | 功能 |
|:---:|:---:|:---|:---|
| 0 | 0 | $Q^n$ | **保持** |
| 1 | 0 | 1 | **置位** |
| 0 | 1 | 0 | **复位** |
| 1 | 1 | $\overline{Q^n}$ | **翻转**（SR禁态改造为功能）|

> [!note] 一次翻转限制
> J=K=1时，在CP=1期间主触发器只能翻转一次（因为翻转后Q与Q̄互换，阻断了进一步触发）。这是一次采用现象在J=K=1条件下的具体表现。

**一次采用 vs 一次翻转的关系：**
- **一次采用现象**：CP=1期间主触发器透明，输入抖动被采样——这是根本机制
- **一次翻转限制**：J=K=1时只翻转一次——这是一次采用现象的一个具体结论

## 四、维持-阻塞 D 触发器（边沿触发，真正解法）

### 核心思想

主从结构在 CP=1 期间主触发器始终透明——这是"一次采用"的根源。维持-阻塞结构的解法：**在 CP↑ 发生后，立即用 Q 的反馈封锁输入通路**，使后续 D 的变化无法传入。

### 6 NAND 门电路结构

```
                    ┌─────────────── 维持反馈 ───────────────┐
                    │                                         │
          S̄d       ↓                                         │
           │   ┌───────┐    ┌───────┐                        │
D  ────────┼──►│  G1   │───►│  G3  ├──── Q ─────────────────┘
           │   │(NAND) │    │(NAND)│     │
CP ────────┤   └───────┘    └───┬──┘     │
           │       │            │阻塞反馈  │
           │       ↓            │◄────────┘
           │   ┌───────┐    ┌───────┐
D̄  ────────┼──►│  G2   │───►│  G4  ├──── Q̄
           │   │(NAND) │    │(NAND)│
          R̄d  └───────┘    └───────┘
```

**两条关键反馈路径：**

| 路径 | 来源 | 去向 | 作用 |
|:---|:---|:---|:---|
| **维持**（Maintain）| Q | G1 输入 | CP↑ 后保持 Q 值不变 |
| **阻塞**（Block）| Q | G2 输入 | CP↑ 后封锁 D̄ 的输入通路 |

### 工作原理逐步推导

**CP↑（上升沿）到来时（以 D=1 为例）：**

1. CP 从 0→1，G1 开启：NAND(D=1, CP=1) → G1输出=0
2. G1输出=0 → G3输入有0 → **Q=1**（输出置位）
3. Q=1 立刻反馈到 G1 → 维持 G1输出=0 锁死（**维持**路径激活）
4. Q=1 同时反馈到 G2 → NAND(Q=1, ...) 使 G2 输入被封锁（**阻塞**路径激活）
5. 此后 D 无论如何变化，G2 的阻塞使信号无法传入输出级

**结论：CP↑ 之后 Q 被双重锁住，输入完全隔离。**

### 工作波形

```
         CP↑ 采样    CP↑ 采样    CP↑ 采样
          ↓           ↓           ↓
CP  ______|‾‾‾‾|______|‾‾‾‾|______|‾‾‾‾|____

D   ______|‾‾‾‾‾‾‾‾‾‾|_____|‾‾‾‾‾‾‾‾‾‾‾‾|___
               ↑ D在CP=1期间变化
               被阻塞，Q不响应！

Q   ______|‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾|_____|‾‾‾‾‾‾‾‾
           D=1被边沿锁住        D=0   D=1
```

**对比：门控D锁存器（透明，有问题）：**

```
CP  ______|‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾|________________

D   ______|‾‾‾‾‾‾‾‾|_____|‾‾‾|________________

Q   ______|‾‾‾‾‾‾‾‾|_____|‾‾‾|________________
           ↑ Q 完全跟随 D（透明），CP=1期间任何D变化都会反映
```

### 异步预置/复位端（$\overline{S_d}$、$\overline{R_d}$）

维持-阻塞D触发器通常带有异步（与CP无关）的置位/复位端：

| $\overline{S_d}$ | $\overline{R_d}$ | 效果 | 优先级 |
|:---:|:---:|:---|:---|
| 0 | 1 | $Q=1$（立即置位，不等CP）| 最高 |
| 1 | 0 | $Q=0$（立即复位，不等CP）| 最高 |
| 1 | 1 | 正常 CP↑ 边沿触发 | — |
| 0 | 0 | 禁止（同SR禁态）| — |

> [!important]
> 异步端优先级**高于** CP 时钟——无论时钟状态，$\overline{S_d}=0$ 立即强制 $Q=1$。这用于上电初始化或强制复位。

### 特性方程

$$Q^{n+1} = D \quad (\text{CP} \uparrow \text{ 时有效})$$

## 五、T 触发器与 T' 触发器

### T 触发器

$$Q^{n+1} = T \oplus Q^n = TQ^n + \overline{T}Q^n$$

| T | $Q^{n+1}$ | 功能 |
|:---:|:---|:---|
| 0 | $Q^n$ | 保持 |
| 1 | $\overline{Q^n}$ | **翻转** |

### T' 触发器（T 恒=1）

$$T=1 \Rightarrow Q^{n+1} = \overline{Q^n}$$

每来一个 CP 时钟沿，输出翻转一次。**N 级 T' 触发器级联 = 异步二进制计数器**（每级分频2）。

> T 触发器是**计数器**的核心构件。

## 六、各类触发器全局对比

| 类型 | 触发方式 | 特征方程 | 禁态 | 主要用途 |
|:---|:---|:---|:---|:---|
| SR 锁存器 | 电平（无CP）| $Q^{n+1}=S_d+\overline{R_d}Q^n$ | $S_dR_d=1$ | 基础存储单元 |
| 门控SR | 电平触发 | 同上，CP=1有效 | 同上 | 简单寄存 |
| 门控D | 电平触发（透明）| $Q^{n+1}=D$ | 无 | 数据锁存 |
| 主从JK | 下降沿（CP↓）| $Q^{n+1}=J\overline{Q}+\overline{K}Q$ | 无（翻转）| 计数/寄存 |
| 维持-阻塞D | 上升沿（CP↑）| $Q^{n+1}=D$ | 无 | 寄存器、DFF |
| T | 边沿触发 | $Q^{n+1}=T\oplus Q^n$ | 无 | 计数器 |

## 七、触发器互转

不同类型触发器之间可以通过**激励函数转换**实现互转：

| 目标 | 已有JK | 方法 |
|:---|:---|:---|
| JK → D | $J=D,\; K=\overline{D}$ | 令K=D̄ |
| JK → T | $J=K=T$ | 两端并联 |
| JK → SR | $J=S,\; K=R$（保留约束）| 直接映射 |
| D → JK | $D=J\overline{Q}+\overline{K}Q$ | 需组合逻辑 |

> [!note] CMOS主从传输门触发器
> 现代数字IC中，高频触发器多用**CMOS主从传输门结构**实现：利用TG的互补开关特性，速度快、功耗极低，是维持-阻塞结构的工艺级等效实现。
