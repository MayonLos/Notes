> **一句话定义**：C++ 标准库的核心，提供容器、迭代器和算法三大通用、高效、可复用的组件，是现代 C++ 开发的必备技能。

> **本页**：STL——`vector`/`map`/`set` 等容器、迭代器、`sort`/`find` 等算法
> **后续**：→ [[内存管理]]（智能指针与容器的配合）

## 三大支柱

| 组件 | 作用 | 示例 |
|:---|:---|:---|
| **容器** | 存储数据的数据结构 | `vector`、`map`、`set` |
| **迭代器** | 类指针对象，统一遍历接口 | `begin()`、`end()` |
| **算法** | 通用操作，作用于迭代器范围 | `sort`、`find`、`transform` |

## 序列容器

### `vector` — 动态数组（最常用）

```cpp
std::vector<int> v = {1, 2, 3, 4, 5};
v.push_back(6);      // 末尾添加 O(1) 均摊
v.pop_back();        // 删除末尾 O(1)
v[0];                // 随机访问 O(1)
```

特点：连续内存、快速随机访问；中间插入 O(n)。

### `list` — 双向链表

特点：中间插入/删除 O(1)，无随机访问。

### `deque` — 双端队列

特点：两端高效、随机访问也可以，但内存非连续。

## 关联容器

### `set` / `map` — 有序（红黑树，O(log n)）

```cpp
std::set<int> s = {3, 1, 4, 1, 5};
// 自动排序、去重：{1, 3, 4, 5}

std::map<std::string, int> age;
age["Alice"] = 25;
for (const auto& [name, years] : age) {  // C++17 结构化绑定
    std::cout << name << ": " << years;
}
```

### `unordered_set` / `unordered_map` — 无序（哈希表，O(1) 均摊）

性能更好，但无序且不支持范围查询。

## 常用算法

```cpp
#include <algorithm>
std::sort(v.begin(), v.end());                    // 排序
auto it = std::find(v.begin(), v.end(), 3);       // 查找
std::count_if(v.begin(), v.end(), [](int x){ return x > 2; }); // 条件计数
std::transform(v.begin(), v.end(), out.begin(), [](int x){ return x*2; }); // 变换
```

## 容器复杂度对比

| 容器 | 随机访问 | 头/尾插入 | 中间插入 | 查找 |
|:---|:---|:---|:---|:---|
| `vector` | O(1) | O(n)/O(1) | O(n) | O(n) |
| `list` | O(n) | O(1) | O(1) | O(n) |
| `deque` | O(1) | O(1) | O(n) | O(n) |
| `set`/`map` | — | O(log n) | O(log n) | O(log n) |
| `unordered_*` | — | O(1) | O(1) | O(1) |
