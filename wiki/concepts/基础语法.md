## 数据类型

### 基本数据类型

| 类型 | 大小 | 范围 | 用途 |
|:---|:---|:---|:---|
| `bool` | 1 字节 | true/false | 布尔值 |
| `char` | 1 字节 | -128 ~ 127 | 单个字符 |
| `int` | 4 字节 | $-2^{31}$ ~ $2^{31}-1$ | 标准整数 |
| `long long` | 8 字节 | $-2^{63}$ ~ $2^{63}-1$ | 极大整数 |
| `float` | 4 字节 | $\pm 3.4 \times 10^{\pm 38}$ | 单精度浮点 |
| `double` | 8 字节 | $\pm 1.7 \times 10^{\pm 308}$ | 双精度浮点 |

### 常用修饰符

- `unsigned`：无符号（范围翻倍，不能表示负数）
- `const`：常量，不可修改
- `static`：静态，改变作用域和存储期

## 变量与初始化

```cpp
int x = 10;         // 传统初始化
int y{5};           // 列表初始化（推荐，C++11+）
int z{};            // 零初始化
```

> [!tip] 推荐使用列表初始化 `{}`
> 列表初始化会阻止窄化转换（如 `int x{3.14}` 会编译报错），更安全。

## 指针与引用

```cpp
int value = 42;
int* ptr = &value;     // 指针：存储地址
int& ref = value;      // 引用：别名

*ptr = 100;            // 解引用修改值
ref = 200;             // 直接通过引用修改
```

> [!warning] 指针 vs 引用
> - 引用必须初始化，不能为空，不能重新绑定
> - 指针可为 nullptr，可重新赋值
> - 优先使用引用（更安全），只有需要可选或重新绑定时使用指针

## 函数

```cpp
// 值传递（拷贝）
void byValue(int x) { x = 99; }   // 不影响外部

// 引用传递（修改原始值）
void byRef(int& x) { x = 99; }    // 影响外部

// const 引用（只读、避免拷贝）
void readOnly(const std::string& s) { /* s 不可修改 */ }
```

### 函数重载

```cpp
int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }
```

### 模板函数

```cpp
template<typename T>
T max(T a, T b) { return a > b ? a : b; }

max(3, 5);       // T = int
max(3.14, 2.71); // T = double
```

## 位运算（对应数字电路逻辑门）

| 操作 | C++ 符号 | 对应逻辑门 |
|:---|:---|:---|
| 按位与 | `a & b` | AND |
| 按位或 | `a \| b` | OR |
| 按位异或 | `a ^ b` | XOR |
| 按位取反 | `~a` | NOT |
| 左移 | `a << n` | — |
| 右移 | `a >> n` | — |
