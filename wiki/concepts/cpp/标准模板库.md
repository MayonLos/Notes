> **一句话定义**：C++ 标准库的核心，提供容器、迭代器和算法三大通用、高效、可复用的组件，是现代 C++ 开发的必备技能。

> [!tip] 学习路径
> **前置**：[[concepts/cpp/基础语法|基础语法]]（模板语法）+ [[concepts/cpp/面向对象|OOP]]（泛型设计思想）
> **本页**：STL——`vector`/`map`/`set` 等容器、迭代器、`sort`/`find` 等算法
> **后续**：→ [[concepts/cpp/内存管理|内存管理]]（智能指针与容器的配合）

## 三大支柱

| 组件 | 作用 | 示例 |
|:---|:---|:---|
| **容器** | 存储数据的数据结构 | `vector`、`map`、`set` |
| **迭代器** | 类指针对象，统一遍历接口 | `begin()`、`end()` |
| **算法** | 通用操作，作用于迭代器范围 | `sort`、`find`、`transform` |

## 序列容器

### `vector` — 动态数组（最常用）

```cpp
std::vector<int> v = {1, 2, 3, 4, 5};
v.push_back(6);      // 末尾添加 O(1) 均摊
v.pop_back();        // 删除末尾 O(1)
v[0];                // 随机访问 O(1)
v.size();            // 元素个数
```

特点：连续内存、快速随机访问、末尾操作高效；中间插入 O(n)。

### `list` — 双向链表

```cpp
std::list<int> lst = {1, 2, 3};
lst.push_front(0);   // 前端插入 O(1)
lst.remove(3);       // 删除值为 3
lst.sort();          // 排序
```

特点：中间插入/删除 O(1)，无随机访问。

### `deque` — 双端队列

```cpp
std::deque<int> dq;
dq.push_front(0);    // 前端 O(1)
dq.push_back(4);     // 末尾 O(1)
```

特点：两端高效、随机访问也可以，但内存非连续。

## 关联容器

### `set` / `map` — 有序（红黑树，O(log n)）

```cpp
std::set<int> s = {3, 1, 4, 1, 5};
// 自动排序、去重：{1, 3, 4, 5}
s.insert(2);          // O(log n)
s.find(3);            // O(log n)

std::map<std::string, int> age;
age["Alice"] = 25;
for (const auto& [name, years] : age) {  // C++17 结构化绑定
    std::cout << name << ": " << years;
}
```

### `unordered_map` / `unordered_set` — 哈希表（O(1) 均摊）

```cpp
std::unordered_map<std::string, int> dict;
dict["key"] = 100;    // 平均 O(1)
```

## 容器选择指南

| 容器 | 访问 | 插入/删除 | 有序 | 适用场景 |
|:---|:---|:---|:---|:---|
| `vector` | O(1) | O(n) | ❌ | 通用，频繁随机访问 |
| `list` | O(n) | O(1) | ❌ | 频繁中间插入 |
| `set` | O(log n) | O(log n) | ✅ | 唯一有序元素 |
| `map` | O(log n) | O(log n) | ✅ | 键值对，有序 |
| `unordered_map` | O(1) 均摊 | O(1) 均摊 | ❌ | 键值对，快速查询 |

## 迭代器

```cpp
std::vector<int> v = {1, 2, 3, 4, 5};

auto it = v.begin();   // 指向第一个元素
std::cout << *it;      // 1
++it;                  // 移动到下一个

auto rit = v.rbegin(); // 反向迭代器，指向最后一个
std::cout << *rit;     // 5

// 范围 for（推荐，底层使用迭代器）
for (int val : v) { std::cout << val; }
```

> [!warning] 迭代器失效
> `vector::push_back` 若触发扩容，会导致**所有迭代器失效**。在遍历过程中修改容器需特别注意。

## 常用算法

```cpp
#include <algorithm>

std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

// 排序
std::sort(v.begin(), v.end());              // 升序
std::sort(v.begin(), v.end(), std::greater<int>());  // 降序

// 查找
auto it = std::find(v.begin(), v.end(), 4);
int cnt = std::count(v.begin(), v.end(), 1);  // 出现次数

// 变换（Lambda）
std::vector<int> sq(v.size());
std::transform(v.begin(), v.end(), sq.begin(),
    [](int x) { return x * x; });
```
