> **一句话定义**：C++ 编程的核心基础，涵盖数据类型、变量、指针与引用、函数等关键知识点，是学习 OOP 和内存管理的前置条件。

## 数据类型

### 基本数据类型

| 类型 | 大小 | 范围 | 用途 |
|:---|:---|:---|:---|
| `bool` | 1 字节 | true/false | 布尔值 |
| `char` | 1 字节 | -128 ~ 127 | 单个字符 |
| `int` | 4 字节 | $-2^{31}$ ~ $2^{31}-1$ | 标准整数 |
| `long long` | 8 字节 | $-2^{63}$ ~ $2^{63}-1$ | 极大整数 |
| `float` | 4 字节 | $\pm 3.4 \times 10^{\pm 38}$ | 单精度浮点 |
| `double` | 8 字节 | $\pm 1.7 \times 10^{\pm 308}$ | 双精度浮点 |

### 常用修饰符

- `unsigned`：无符号（范围翻倍，不能表示负数）
- `const`：常量，不可修改
- `static`：静态，改变作用域和存储期

## 变量与初始化

```cpp
int x = 10;         // 传统初始化
int y{5};           // 列表初始化（推荐，C++11+）
int z{};            // 零初始化

// 静态变量：程序整个生命周期存在
static int counter = 0;
```

> [!tip] 推荐使用列表初始化 `{}`
> 列表初始化会阻止窄化转换（如 `int x{3.14}` 会编译报错），更安全。

## 指针与引用

### 指针

```cpp
int value = 42;
int* ptr = &value;     // & 取地址

std::cout << *ptr;     // * 解引用，输出 42
ptr->member;           // 访问成员（等价于 (*ptr).member）

int arr[5] = {1,2,3,4,5};
int* p = arr;          // 数组名退化为指针
```

> [!warning] 悬垂指针（Dangling Pointer）
> 指向已释放内存的指针会导致未定义行为。删除后必须置 `nullptr`：
> ```cpp
> int* ptr = new int(42);
> delete ptr;
> ptr = nullptr;    // 必须！防止二次释放或误用
> ```

### 引用

引用是别名，创建后不能改变指向：

```cpp
int value = 42;
int& ref = value;
ref = 100;        // 修改原变量，value 变为 100
```

## 函数

### 参数传递方式

```cpp
// 值传递：复制，不影响原变量
void increment(int x) { x++; }

// 引用传递：修改原变量
void increment(int& x) { x++; }

// 常量引用：高效传大对象，不修改
void print(const std::string& str) { std::cout << str; }

// 默认参数
int power(int base, int exp = 2) { return base ^ exp; }
```

### 函数重载

同一作用域，名称相同但参数列表不同：

```cpp
int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }
```

## 数组与字符串

```cpp
// 数组
int arr[] = {1, 2, 3};    // 自动推断长度 = 3

// C++ string（推荐，自动管理内存）
std::string s = "Hello";
s += " World";
s.length();               // 11
```

> [!warning] C 风格字符串的危险
> `char[]` 易导致缓冲区溢出，优先使用 `std::string`。

## 控制流

```cpp
// 范围 for（推荐）
std::vector<int> v = {1, 2, 3, 4, 5};
for (int num : v) { std::cout << num; }

// 传统 for
for (int i = 0; i < 5; ++i) { ... }
```

## 模板基础

```cpp
template<typename T>
T max_value(T a, T b) { return (a > b) ? a : b; }

max_value(5, 3);      // T 推导为 int
max_value(5.5, 3.2);  // T 推导为 double
```
