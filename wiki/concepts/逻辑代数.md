> **一句话定义**：逻辑代数（布尔代数）是研究逻辑变量（仅取 0/1 两值）之间运算关系的数学工具，是数字电路分析与设计的理论基础。

---

## 一、基本逻辑运算

| 运算 | 符号 | 表达式 | 描述 |
|:---:|:---:|:---:|:---|
| 与（AND） | · | $F = A \cdot B$ | 全1出1，有0出0 |
| 或（OR） | + | $F = A + B$ | 有1出1，全0出0 |
| 非（NOT） | ¯ | $F = \bar{A}$ | 取反 |
| 异或（XOR） | ⊕ | $F = A \oplus B$ | 相异出1 |
| 同或（XNOR） | ⊙ | $F = A \odot B$ | 相同出1 |

> [!important]
> 异或与同或互为反运算：$A \oplus B = \overline{A \odot B}$

---

## 二、常用逻辑门

| 门名称 | 符号 | 逻辑表达式 | 说明 |
|:---:|:---:|:---|:---|
| 与非门 | NAND | $F = \overline{A \cdot B} = \bar{A} + \bar{B}$ | 有 0 出 1 |
| 或非门 | NOR | $F = \overline{A + B} = \bar{A} \cdot \bar{B}$ | 全 0 出 1 |

> [!info] 新发现
> NAND 门和 NOR 门均为**功能完备集**：仅用 NAND（或仅用 NOR）即可实现任意逻辑函数。

---

## 三、基本公式与常用公式

### 基本定律

| 定律 | 公式 |
|:---|:---|
| **交换律** | $A+B = B+A$；$AB = BA$ |
| **结合律** | $(A+B)+C = A+(B+C)$ |
| **分配律** | $A(B+C) = AB+AC$；$A+BC = (A+B)(A+C)$ |
| **德摩根定律** | $\overline{AB} = \bar{A}+\bar{B}$；$\overline{A+B} = \bar{A}\cdot\bar{B}$ |
| **0-1 律** | $A+0=A$；$A\cdot1=A$；$A+1=1$；$A\cdot0=0$ |
| **互补律** | $A+\bar{A}=1$；$A\cdot\bar{A}=0$ |
| **幂等律** | $A+A=A$；$A\cdot A=A$ |
| **双重否定** | $\bar{\bar{A}}=A$ |

> [!important]
> **德摩根定律**是逻辑化简和门电路转换的核心工具：取反时，将与变或、或变与，并对每个变量取反。

### 常用等式

| 名称 | 公式 |
|:---|:---|
| 吸收律 | $A + AB = A$ |
| 并项 | $AB + A\bar{B} = A$ |
| 消因子 | $A + \bar{A}B = A + B$ |
| 冗余律 | $AB + \bar{A}C + BC = AB + \bar{A}C$ |

## 四、逻辑函数的建立

1. **确定输入变量**
2. **确定输出变量**
3. **列真值表**：枚举所有输入组合（$n$ 变量共 $2^n$ 行）
4. **写逻辑表达式**：从真值表中输出为 1 的行写最小项之和（SOP）
5. **化简**：见 [[逻辑函数化简]]
